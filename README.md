INSTRUCTIONS AND REQUIREMENTS TO MINE EDEN COIN
Download cpuminer-multi from the link - https://github.com/tpruvot/cpuminer-multi/releases
Configure the conf file accoording to the instruction written in GetStarted to mine Eden Configure

Use user - Your Eden Wallet Address

To download Eden Wallet visit - https://edenwallet.app or directly from Play Store.
